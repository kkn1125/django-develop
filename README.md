# 스케줄 공유 서비스

업무 스케줄이나 간단한 일정 조율을 위한 스케줄 공유 서비스

## Overview

학업과 회의 또는 간단한 일정 조율을 할 때 캘린더를 공유하면서 실시간으로 변경되는 일정을 빠르게 조율하기 위해 서비스를 설계.
